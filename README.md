# LearnFi - Remote Learning Tokens 🗂️

## Overview

LearnFi is a decentralized learning platform built on the Stacks blockchain that enables students to purchase digital course access using cryptocurrency tokens. The system provides a secure, transparent, and efficient way to manage online course payments and access rights.

## Features

### 🎓 Course Management
- Create and manage digital courses with customizable pricing
- Set course availability and enrollment limits
- Track course statistics and revenue

### 💰 Token-Based Payments
- Purchase course access using STX tokens
- Automatic enrollment upon successful payment
- Secure payment processing with built-in refund mechanisms

### 🔐 Access Control
- Verify student enrollment before granting course access
- Time-based access management
- Instructor permissions for course content management

### 📊 Analytics & Tracking
- Monitor course enrollment numbers
- Track payment history and revenue
- Student progress tracking capabilities

## Smart Contracts

### 1. Course Manager Contract
- Handles course creation and management
- Manages instructor permissions
- Tracks course metadata and pricing

### 2. Payment Processor Contract  
- Processes STX payments for course access
- Manages student enrollments
- Handles refunds and payment disputes

## System Architecture

The LearnFi platform consists of two main smart contracts working together:

1. **Course Management Layer**: Handles course creation, pricing, and metadata
2. **Payment Processing Layer**: Manages student payments and access rights

## Getting Started

### Prerequisites
- Stacks wallet with STX tokens
- Access to Stacks blockchain network

### For Students
1. Browse available courses
2. Select desired course and review pricing
3. Make payment using STX tokens
4. Gain immediate access to course content

### For Instructors
1. Register as course instructor
2. Create new courses with pricing and metadata
3. Monitor enrollments and revenue
4. Manage course content and access
